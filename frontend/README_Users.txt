# Excellence Coaching Hub - Installation Instructions

## Quick Installation

**Simply double-click "Easy Install.bat" to install the application.**

## What This Does

This installer will:
1. Build the Excellence Coaching Hub application
2. Install it to your computer
3. Create a desktop shortcut for easy access
4. Make the app available in your Start Menu

## Requirements

- Windows 10 or 11
- Flutter SDK installed and added to your PATH
- Internet connection for the initial build

## After Installation

Once installed, you can launch Excellence Coaching Hub by:
- Clicking the desktop shortcut
- Searching for it in your Start Menu
- Running it from the installation folder

## Troubleshooting

If you encounter any issues:
1. Make sure Flutter is properly installed
2. Check that you have internet connection
3. Try running the installer as Administrator

## Installation Location

The app is installed to: `%LOCALAPPDATA%\ExcellenceCoachingHub`

## Uninstall

To uninstall, simply delete the installation folder and the desktop shortcut.
