# Excellence Coaching Hub - Web Assets

This directory contains the web-facing components for the Excellence Coaching Hub application.

## File Structure

- `index.html` - The main Flutter web application entry point
- `landing.html` - Landing page with app promotion and download options
- `download.html` - Detailed download page with build instructions for Android and iOS
- `manifest.json` - Web app manifest for PWA functionality
- `favicon.png` - Site favicon
- `icons/` - App icons for various sizes
- `downloads/` - Directory containing build instructions and documentation

## Access Points

- **Main App**: Access the full Flutter web application through `index.html`
- **Landing Page**: Promotional page with download options at `landing.html`
- **Download Page**: Detailed build instructions at `download.html`
- **Build Instructions**: Comprehensive build guide at `downloads/index.html`

## Purpose

This web directory serves dual purposes:
1. Host the Flutter web application
2. Provide an attractive landing/download page for mobile app distribution

The mobile app needs to be built from source as it's a self-hosted educational platform.
