Excellence Coaching Hub Downloads

Build and install instructions for Android and iOS

Prerequisites

  • Flutter SDK (3.0.0 or higher)
  • Android Studio or VS Code with Flutter extensions
  • Android SDK for Android builds
  • Xcode for iOS builds (macOS only)
  • Git for cloning the repository

Getting Started

First, clone the repository and navigate to the frontend directory:

# Clone the repository (replace with actual repo URL)
git clone https://github.com/your-username/excellence-coaching-hub.git
# Navigate to frontend
cd excellence-coaching-hub/frontend
# Get dependencies
flutter pub get

Building for Android

Generate code and build the Android APK:

# Generate code (for Freezed models)
flutter pub run build_runner build
# Build the release APK
flutter build apk --release
# For a universal APK (recommended)
flutter build apk --split-per-abi --release

The APK file will be located at: build/app/outputs/flutter-apk/app-release.apk

Building for iOS

Build the iOS IPA (requires macOS and Xcode):

# Generate code (for Freezed models)
flutter pub run build_runner build
# Open iOS project in Xcode (macOS only)
cd ios && open Runner.xcworkspace
# In Xcode: select your development team and update bundle identifier
# Build the release IPA
flutter build ios --release

The IPA file will be located in: build/ios/ipa/

Configuration

Make sure to set up your environment variables before building:

  1. Create a .env file in the frontend directory
  2. Add the API base URL:
# Content of .env file
API_BASE_URL=https://your-backend-api.com/api
